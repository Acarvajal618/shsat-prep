Unit 10_2

Circumference from Rectangle Dimensions

Use the rectangle's perimeter to uncover the circle's diameter, then apply C = πd.

Strategy

The Approach

The radius equals the rectangle's short side, so the diameter equals the long side. Express the perimeter in terms of the diameter, solve for d, then compute circumference using C = πd.

  • 🔍Circle is inscribed in a rectangle/square.
  • 🔍Perimeter is given instead of a side length.
  • 🔍The problem explicitly labels a radius reaching the opposite side.

The Problem

P is the center of the circle shown above. AB is a diameter. The perimeter of rectangle ABCD is 72 cm. PQ is a radius, Q is on CD. What is the circumference of the circle?

Step-by-Step Solution

  1. 1
    Relate the rectangle to the circle.
    If AB is a diameter, then BC (the short side) equals the radius r.
  • 2
    Write the perimeter in terms of the diameter.
    Perimeter = 2(long + short) = 2(d + d/2) = 3d.
  • 3
    Solve for the diameter.
    3d = 72 ⇒ d = 24 cm.
  • 4
    Compute circumference.
    C = πd = 24π.
    ✅
    Answer: G → 24π cm
    Short side = radius = d/2, so perimeter = 3d = 72 ⇒ d = 24 ⇒ C = 24π.

    Common Mistakes

    • ❌
      Dividing 72 by 2and calling 36 the diameter ignores that the short side is only half the diameter.
    • ❌
      Using C = 2πr with the wrong radiusIf you plug in 36 for r, you get 72π — way too large.
    • ✅
      Label the rectangle sidesAnnotate the diagram with "d" and "d/2" before writing the perimeter equation.
