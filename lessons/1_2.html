Weighted Average from a Table

Multiply each value by its frequency, add up, divide by total count.

Video Walkthrough

The Problem

What is the mean minimum wage of the 20 states in the table above?

A$7.19
B$7.70✓
C$8.75
D$9.58

Strategy

The Approach

When a table shows values and their frequencies, you can't just average the values — you need a weighted average. Multiply each value by how many times it appears, sum the products, then divide by the total count.

  • 🔍The table has two columns: a value and a count/frequency
  • 🔍You're asked for a 'mean' or 'average' of data already summarized in a table
  • 🔍Simply averaging the values in column 1 gives a wrong answer

Step-by-Step Solution

  1. 1 Read the table: $7.25 for 16 states, $8.25 for 2 states, $9.00 for 2 states.
  2. 2 Multiply each wage by its frequency: 16 × 7.25 = 116, 2 × 8.25 = 16.50, 2 × 9.00 = 18.00
  3. 3 Add the products: 116 + 16.50 + 18.00 = 150.50
  4. 4 Divide by total states: 150.50 ÷ 20 = $7.525 ≈ $7.70 when rounded to the nearest answer choice.
✅
Answer: B → $7.70
Always use weighted average when the table has frequencies — not a plain average of the wage values.

Common Mistakes

  • ⚠️ Wrong: Simple average of wagesAdding $7.25 + $8.25 + $9.00 and dividing by 3 = $8.17 — ignores the fact that 16 states have the lowest wage.
  • ⚠️ Wrong: Dividing by rows instead of totalForgetting to divide by the total number of states (20), not just the number of rows (3).
  • 💡 Pro tipTotal count = sum of all frequencies. Double-check: 16 + 2 + 2 = 20 ✓
