Unit 2_7 · Plug & Solve

Solve a System Implied by Constraints

The constraints define the variable values. Extract them, then solve.

The Problem

If x(b − c) = y + x and 2b = 3c = 7, what is the value of y/x?

A 1/7
B 1/6 ✓ answer
C 1/3
D 5/7

The Strategy

Core idea

When constraints like "2b = 3c = 7" define variables indirectly, solve for each variable from the constraint first. Then substitute into the main equation and simplify to find the target ratio or value.

When you'll see this

  • 🔍"2b = 3c = 7" means each expression equals 7 — use that to find b and c separately.
  • 🔍The question asks for a ratio y/x, not an absolute value — look for cancellation.
  • 🔍Work the constraints first, before touching the main equation.

Step-by-Step Solution

  1. 1
    Extract b and c from the constraints.
    2b = 7 → b = 7/2 = 3.5
    3c = 7 → c = 7/3
  2. 2
    Calculate b − c.
    7/2 − 7/3 = 21/6 − 14/6 = 7/6
  3. 3
    Substitute into the main equation.
    x(7/6) = y + x
  4. 4
    Isolate y.
    7x/6 − x = y
    7x/6 − 6x/6 = y
    x/6 = y
  5. 5
    Write the ratio.
    y/x = (x/6)/x = 1/6
✅
y/x = 1/6 → Answer: B
Check: if x = 6, then y = 1. x(b−c) = 6(7/6) = 7. y + x = 1 + 6 = 7 ✓

Common Mistakes

  • ❌
    Treating 2b = 3c = 7 as b = c = 7 b and c are different. 2b = 7 gives b = 3.5; 3c = 7 gives c ≈ 2.33. They're not equal to each other or to 7.
  • ❌
    Forgetting to find a common denominator for b − c Computing 3.5 − 2.33 ≈ 1.17 by decimal is imprecise. Use fractions: 7/2 − 7/3 = 7/6 exactly.
