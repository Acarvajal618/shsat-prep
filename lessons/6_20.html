Unit 6_20

Theme Park Pass Probability

Always sum ALL rows before computing probability.

Strategy

The Approach

Find the grand total of all park passes sold. Then divide the 3-Day 3-Parks count by that grand total to get the probability.

  • 🔍Total = 84 + 56 + 45 + 49 + 66 = 300.
  • 🔍3-Day 3-Parks count = 66.
  • 🔍Probability = 66/300 = 22%.

The Problem

The table above shows the number of times that different types of park passes were purchased at a theme park. Based on this information, what is the probability of a customer purchasing a 3-Day, 3-Parks pass?

Pass TypePurchases
1-Day, 1-Park84
1-Day, 2-Parks56
1-Day, 3-Parks45
2-Day, 2-Parks49
3-Day, 3-Parks66
A22%✓
B37%
C50%
D66%

Step-by-Step Solution

  1. 1
    Sum all pass types.
    84 + 56 + 45 + 49 + 66 = 300
  2. 2
    Probability of 3-Day, 3-Parks.
    66 / 300 = 0.22 = 22%
✅
Answer A — 22%
66/300 = 0.22 = 22% ✓

Common Mistakes

  • ❌
    Raw count as answerD = 66% — that's the raw purchase count, not a probability.
  • ❌
    Partial totalDividing by fewer than all 5 pass types gives an inflated probability.
  • ✅
    Right moveAlways sum ALL rows first: 84+56+45+49+66 = 300.
