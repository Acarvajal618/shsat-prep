Unit 9_5

Salad Bar Pricing

Pay the base price for the first portion, then add the overflow cost.

Strategy

The Approach

Separate the base portion (first 8 ounces) from the extra ounces. Keep the fixed cost and the variable cost distinct, then add them.

  • 🔍"First X units cost $Y" followed by an "each additional" statement.
  • 🔍Target size is larger than the base size.
  • 🔍Use two parts: base price + extra-ounces price.

The Problem

Sidney is eating lunch at a salad bar where an 8-ounce salad costs $2.49. Each additional ounce costs $0.50. How much would a 16-ounce salad cost?

A$5.49
B$6.49✓
C$6.99
D$7.49

Step-by-Step Solution

  1. 1
    Identify the base portion.
    The first 8 ounces always cost $2.49.
  • 2
    Find the overflow ounces.
    16 − 8 = 8 extra ounces.
  • 3
    Compute the extra cost.
    8 × $0.50 = $4.00.
  • 4
    Add base and extra.
    $2.49 + $4.00 = $6.49.
    ✅
    Answer: B → $6.49
    Base $2.49 + 8 extra ounces at 50¢ each = $6.49 ✓

    Common Mistakes

    • ❌
      Charging the extra rate for all 16 ounces($16 × 0.50 = $8) ignores the cheaper base portion.
    • ❌
      Forgetting to add the base $2.49and returning $4.00 as the answer.
    • ✅
      Use two bucketsAlways compute (base cost) + (overflow cost) separately so you don't double-count.
